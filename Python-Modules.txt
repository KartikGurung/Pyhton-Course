# What is Module :
A Python Module is a file containing variables, class definition, satements, &functions related to a particular task.
The major feature of a module is that the content of it can be used in other programs without having to rewrite or recreate them.

# Structure of Module :
    1. docstring : triple quoted string for document purpose (""" """)
    2. variaables & constants : 
    3. classes : templates or blueprints
    4. objects : instances of class
    5. satements : instructions
    6. functions : for performing a task.
